  • Data Entry Form with Validation

    In this project, I created a data entry form with fields like full name, phone number, email, address, and dropdown selections.   • To-Do List Web App

    This is a dynamic to-do list application created using HTML, CSS, and JavaScript. It enables users to add, update, mark, and delete tasks. The app uses DOM manipulation and JavaScript event handling to provide a real-time interactive experience. Through this project, I learned how to manage data states in the frontend, handle user inputs, and improve UI responsiveness. The layout is kept simple and intuitive to ensure ease of use.
